# StayEasy

## Description

StayEasy is a web application for managing listings.  
Users can create, edit, view, and delete listings with details such as title, description, price, location, country, and image.  
The app features a clean, responsive design, making it easy to use on both desktop and mobile devices.

The project is still in progress, but the current version has a working CRUD system and a professional layout.

## Features (Completed)

- Add new listings with title, description, image, price, location, and country
- Edit existing listings with pre-filled forms
- View listing details in a clear split-layout card
- Delete listings
- Responsive design using Bootstrap 5
- Clean card-based UI

## Planned Features

- User login and authentication
- Multiple images per listing
- Search and filter listings by location or price
- Ratings and reviews for listings
- Image preview while creating/editing listings

## Tech Stack

- **Backend:** Node.js, Express.js
- **Frontend:** EJS templates, Bootstrap 5
- **Database:** MongoDB (Mongoose)
- **Version Control:** Git, GitHub
